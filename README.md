# my-shop

React + Vite 프로젝트입니다.

## 기술 스택

- **Nodejs 20** - 기본 개발 환경
- **React 19** - 사용자 인터페이스 라이브러리
- **Vite** - 빠른 개발 서버 및 빌드 도구

## 개발 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 빌드
npm run build

# 빌드된 프로젝트 미리보기
npm run preview
```

## 개발 가이드

이 프로젝트는 **JavaScript**를 사용하며 다음 스타일을 권장합니다:

- 함수형 컴포넌트와 React 훅 사용
- 화살표 함수로 컴포넌트 작성
